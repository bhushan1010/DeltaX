Lead Listing Screen for LeadFlow Pro - Professional automotive lead management system. Features a sortable paginated table with multi-select checkboxes, inline status dropdowns, quick actions (call/email/edit/delete), virtual scrolling for 1000+ rows, global search, filters (status/source/priority/agent/date), saved presets (My Leads, Unassigned, Hot Leads), bulk actions, real-time "Being edited by [User]" indicators, live new-lead notification badge. Uses HSR Blue (#0066CC) as primary color, clean Inter font, professional automotive dashboard aesthetic.

**DESIGN SYSTEM (REQUIRED):**
- Platform: Web, Desktop-first
- Palette: Primary Action (#0066CC), Secondary Action (#64748B), Success (#10B981), Warning (#F59E0B), Error (#EF4444), Background (#FFFFFF), Surface (#F8FAFC), Text Primary (#1E293B), Text Secondary (#64748B), Border (#E2E8F0)
- Styles: Softly rounded (8px), Whisper-soft shadow, Clean Inter typography

**PAGE STRUCTURE:**
1. **Header:** Sticky navigation bar with LeadFlow Pro logo, global search (wide input), user avatar dropdown, real-time notification bell badge
2. **Filters Panel:** Collapsible sidebar with filter controls (status, source, priority, agent, date range), save preset buttons, bulk action toolbar (appears when selections made)
3. **Primary Content Area:** 
   - Toolbar: New Lead button (primary), export dropdown, view presets
   - Table: Virtual scrolling table with columns for checkbox, lead info (name, avatar, source badge), contact info, assigned agent, status (inline dropdown), priority, last activity, actions (call/email/edit/delete)
   - Row highlighting: Blue 5% hover, selected row Blue 10% background
   - Real-time indicators: "Being edited by [User]" tooltip on row hover, live badge for new leads
4. **Footer:** Status bar showing total leads, selected count, page info
